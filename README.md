# PPTX Ripper
A desktop productivity tool for students and researchers. It automates the process of converting PPTX presentations into study notes.

## Features
- Batch Processing of PPTX files
- OCR integration with Tesseract
- Flexible output options
- Media extraction for powerpoint images

## How to Use
Requires Python 3.x

Clone the repository with

`git clone https://github.com/kplunkett05/PPTX-Ripper.git`

Install dependencies

`pip install -r requirements.txt`

Run the application with

`python gui_app.py`
